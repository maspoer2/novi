My first web

Ini dengan menggunakan visual studio code di Linux Debian 13.

Menggunakan Brave Browser.

Untuk menjadi seorang software engineer di bidang pengembangan web. Berikut adalah hal-hal yang penting untuk diketahui:

  1. Belajar konsep HTML sampai ke Form
  2. Belajar bagaimana mengakses DOM dan memberi nama ke variable
  3. Nah variable itu nanti yang akan kita manipulasi dengan Java Script
  4. Cara mudah membuat onclick itu langsung ditulis di kode html pada bagian class-nya dengan cara onclick="fungsi()"
